Interweb.WTF is a privacy-focused and user-centric URL shortening service that allows you to shorten URLs into easily shareable links, expand and remove tracking parameters from shortlinks from other URL shortening services, and clean tracking parameters from any URL.

## API Documentation

<Alert icon={<IconInfoCircle />} color="teal">
  The Interweb.WTF API is currently in beta and subject to change.
</Alert>

The Interweb.WTF API is a simple API that allows you to interact with the most basic
features of the Interweb.WTF service.
