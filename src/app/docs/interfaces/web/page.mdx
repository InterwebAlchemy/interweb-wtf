## Web Interface

The Web Interface is the main interface for Interweb.WTF.

It is a web-based interface that allows you to interact with Interweb.WTF from any browser.

With the Web Interface, you can:

- [Create a WTF Link](/docs/shorten)
- [Preview a WTF Link](/docs/concepts/wtf-links)
- [Expand a shortlink](/docs/expand)
- [Clean a URL](/docs/clean)
